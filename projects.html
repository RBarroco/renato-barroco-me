Ames House - Regression

The goal of this project was to apply Exploratory Data Analysis, Features Engineering, and to use Regression in Machine Leaning tools.

  • Dataset: The dataset is the famous Ames Housing, which has 82 features (columns) and 2930 records (rows).
  • Tools:
    • 1) Ridge Regression, K-Nearest Neighbors, Support Vector Regression, Decision Trees, Neural Networks, Random Forest, Gradient Boosting.
    • 2) A dicussion about standarization of data, in linear regression vs. more complex models.
    • 3) Polynomial features in different degree, and why to use them.
    • 4) Split data between training set and testing set.
    • 5) Cross validation method, using different regressions methods inclusive: GridSearchCV, RandomizedSearchCV.
    • 6) Exploratory data analysis to find outliers, to understand strong vs. weak correlation between predictor variables themselves and between predictor variables vs. target variable.
    • 7) Feature Engineering (One hot Encode) based on categorical unordered, handling categorical ordered data through mapping technique, and handling 'NaN' data.
    • 8) Correlation matrix, heatmap of correlation matrix, overall multi- and uni-variate analysis.

Web API quandl.com

This is a project about web API of the website quandl.com, a timeseries data, and Exploratory Data Analysis.

  • Dataset: I chose the Big Mac Index, because it is a great set of data to explain the concept of purchasing power parity between countries and the consequences of inflation in the day to day of our lives: Big Mac Index Data. I had to work on the data, because it's a type of series data and, as usual, it is difficult to find for all countries, in all date data. Which made the project so much more exciting because I had to be creative. I'm planning to make an article out of this project.

    • Tools:
      • 1) Libraries like Requests, json, Pandas, Numpy, matplotlib, Seaborn, scipy, and relational database concepts.
      • 2) Data Structures concepts, functional programming, error handling.
      • 3) Features Engineering, data wrangling, timeseries data.
      • 4) Correlation, Correlation heatmap, uni-variate and multi-variate analysis.
      • 5) Insights from data, a creative approach towards finding better conclusions.
      • 6) Purchasing Power Parity, inflation, world currency system, interest rates, economics of real life.
